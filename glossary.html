Key Performance Indicators (KPIs)

KPIUser Count

Total number of unique registered users

Table: public.users

View: users

Measure type: count_distinct

SQL: ${id}

KPITotal Revenue

Sum of all order sale prices

Table: public.orders

View: orders

Measure type: sum

SQL: ${sale_price}

Format: usd

Links: Revenue DashboardFinance Deep Dive

KPIOrder Count

Total number of orders

Table: public.orders

View: orders

Measure type: count

Links: Orders Dashboard

KPIAverage Order Value

Average revenue per order

Table: public.orders

View: orders

Measure type: average

SQL: ${sale_price}

Format: usd

KPIConversion Rate

Ratio of completed orders to total orders

Table: public.orders

View: orders

Measure type: number

SQL: 1.0 * ${total_orders_completed} / NULLIF(${order_count}, 0)

Format: percent_2
